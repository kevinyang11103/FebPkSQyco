# 自习室预约管理系统 java384

本项目是一款前后端分离的自习室预约管理系统，旨在帮助管理员高效管理自习室资源，同时方便学生进行在线预约与时间管理。

## 技术栈

- **后端：**
  - Spring Boot
  - MyBatis
- **前端：**
  - Vue 2
  - ElementUI
- **数据库：**
  - MySQL 8.0

## 功能模块

### 系统角色

- 管理员
- 学生

### 功能概述

#### 管理员

- 管理学生信息
- 管理教室信息
- 查看学生预约情况

#### 学生

- 选择预约时间和教室
- 查看预约记录
- 取消预约
- 修改个人信息

## 运行环境

- 后端：Java 1.8+，Spring Boot 环境配置
- 前端：Node.js 环境，Vue CLI 工具
- 数据库：MySQL 8.0

## 部署步骤

1. 搭建开发环境，确保 Java、Node.js、MySQL 等依赖软件已正确安装。
2. 克隆后端代码，按照 `README.md` 说明进行项目构建与部署。
3. 克隆前端代码，使用 Vue CLI 命令行工具进行项目构建与部署。
4. 配置数据库，执行 SQL 脚本创建相关数据表与初始化数据。
5. 运行后端服务，确保 API 接口可用。
6. 运行前端服务，与后端服务进行交互。

## 目录结构

```
项目根目录/
```

- 前后端分离，便于维护与扩展
- 使用 Spring Boot 与 Vue 2，保证系统的高效与稳定性
- 提供完整的用户角色权限管理，增强系统安全性
- 支持预约管理与个人信息维护，实现资源的高效利用
